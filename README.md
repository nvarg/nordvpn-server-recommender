# Nord VPN Server Recommender

## Usage

```
usage: nordvpn-server-recommender.sh [-hu] [-c <country code>] [-t <technology>] [-g <group>]

-c <country code>     only recommends servers located in that country
-c list               displays a list of the available country codes
                      
-t <technology name>  only recommends servers with that technology available.
-t list               displays a list of the available technologies
                      
-g <group identfier>  only recommends servers in that group
-g list               displays a list of the available group identifiers
                      
-u                    displays usage
                      
-h                    displays this help menu
```
